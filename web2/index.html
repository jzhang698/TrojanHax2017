Multilayer Perceptrons

In Machine Learning, the Perceptron is an algorithm for classifying binary data. Given large sets of data, it can gradually learn to divide objects into two distinct groups based on certain characteristics. With more data, it can more accurately differentiate between the groups by decision boundaries.

For Spring 2017 Trojan Hacks, our team is implementing a deep learning algorithm to diagnose abnormal heart complications. This algorithm can be used to help save lives and make better predictions than doctors.

It utilizes a deep learning network with TensorFlow to make these predictions.

Background Information

We implemented a simple deep neural network to create a multilayer perceptron. By training this network with known patient information, we were able to predict (given a patient's data) whether they have a heart abnormality. Here is how we did it:

Source of Data

Our training and testing points were obtained from UCI's Database of MPI Testing data. It included 267 distinct patients.

What Is an MPI?

The imaging data was obtained through Single Photo Emission Computed Tomography (SPECT), which is a type of Myocardial Perfusion Imaging.

Classification

After training the algorithm with the data set, it was able to classify the data into 2 groups - normal and abnormal.

Our Results

The algorithm was able to classify over 80% of the test inputs correctly after being trained on 120 data points.

Brief Explanation of the Algorithm

The basic (linear)perceptron algorithm takes in an initial data point and draws a divider arbitrarily, as that point is the first to be plotted. After the second data point is drawn, it adjusts the divider accordingly. For example, if the first and second points are in opposite groups, then the divider goes between them in some way. The slope of the graph is the weight of the data points and the y-intercept is the bias. The multilayer perceptron is an extrapolation of this concept to nonlinear algorithms inside neural networks.

Perceptron Learning Curve

A neural network is only as good as the data points that it's trained upon. This graph has a very fast and steep learning curve, so one can observe that it has reached its threshold of diminishing returns.
